# ValhallaGYM

Valhalla GYM es una página moderna para un gimnasio, diseñada para mostrar servicios, atraer clientes y ofrecer una experiencia visual atractiva enfocada en fitness.

Este proyecto fue desarrollado como práctica de desarrollo web frontend, aplicando conceptos de diseño responsivo y estructura de interfaces.


# 🧱 Tecnologías utilizadas
HTML5
CSS3
Java
JavaScript

# Características

- Diseño moderno enfocado en fitness
- Adaptable a dispositivos móviles
- Secciones de servicios y planes
- UI atractiva

# 🎯 Objetivo del proyecto

El objetivo es simular una página real de gimnasio que:

- Presente servicios de manera clara
- Genere interés en potenciales clientes
- Practique diseño UI/UX
- Refuerce habilidades en frontend
